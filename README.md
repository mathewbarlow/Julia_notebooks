# Julia_notebooks
Jupyter notebooks for Julia code

<b> run_SpeedyWeather_Julia_Colab.ipynb </b>

This Julia notebook runs the SpeedyWeather model in Google Colab. The runtime needs to be set to Julia in Colab. I don't know any Julia - the aim here is just to be able to get a quick sense of the model before committing to installing another language. Downside: loading the packages takes around ten minutes. Upside: can try out the model without having to install Julia on your own computer. The image shows mid-level vorticity from day 10 of a T42L8 aquaplanet run.
